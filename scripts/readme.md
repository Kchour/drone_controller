# Info
Make sure all script here are executable via `chmod +x *.sh`

All scripts must be run from this directory

First replace the device name in both the setup and clean up script with your device name

Run the setup scripts to route data to aliases: `$ ./setup_drones.sh`  

Run your flight controllers, tests, etc...

When you're finished, run clean up scripts: `$ ./cleanup_drones.sh`
